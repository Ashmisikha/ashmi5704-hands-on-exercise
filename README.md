# 🎓 Student Course Portal

A responsive and interactive **Student Course Portal** developed using **Angular 20** as part of the **Cognizant Digital Nurture 5.0 Angular Hands-On Exercise**.

## 🚀 Features

- Student Dashboard
- Course Listing
- Course Enrollment
- Student Profile Management
- Reactive Forms & Validation
- Angular Routing
- Angular Services
- NgRx State Management
- Responsive User Interface

## 🛠️ Technologies Used

- Angular 20
- TypeScript
- HTML5
- SCSS
- RxJS
- NgRx
- Angular CLI

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/Ashmisikha/ashmi5704-hands-on-exercise.git
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
ng serve
```

Open:

```
http://localhost:4200
```

## 📚 Learning Outcomes

This project demonstrates:

- Component-based architecture
- Routing and Navigation
- Reactive Forms
- Services and Dependency Injection
- State Management with NgRx
- Responsive UI Development
